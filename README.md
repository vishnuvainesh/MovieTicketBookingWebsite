# 🎬 CineVerse - Premium Movie Ticket Booking Website

CineVerse is a modern, responsive, and interactive movie ticket booking web application built using HTML, CSS, and JavaScript. It provides users with a seamless cinema booking experience, including movie browsing, theater selection, seat booking, secure payment simulation, booking confirmation, and user dashboard management.

---

## 🚀 Features

### 🏠 Home Page
- Dynamic hero movie slider
- Featured movies section
- Upcoming movies carousel
- Special offers showcase
- Fully responsive navigation bar

### 🎥 Movies Section
- Browse all available movies
- Filter movies by:
  - Genre
  - Language
  - Rating
  - Format (IMAX, 3D, 4DX, etc.)
- Movie details page

### 🎭 Theater & Showtime Selection
- Search theaters by city or name
- Select preferred date
- View available showtimes

### 💺 Seat Booking System
- Interactive seat map
- Multiple seat categories:
  - VIP
  - Premium
  - Regular
- Real-time seat selection
- Automatic price calculation

### 💳 Payment Gateway Simulation
- Credit/Debit Card Payment
- UPI Payment
- Net Banking
- Digital Wallets
- Promo Code Support

### 🎫 Booking Confirmation
- Digital ticket generation
- Booking ID creation
- QR Code display
- Ticket download option

### 👤 User Dashboard
- Booking history
- Profile management
- Saved payment methods
- Favorite movies section

### 🔐 Authentication
- User Login Page
- User Registration Page
- Password visibility toggle

### 📱 Responsive Design
- Desktop optimized
- Tablet friendly
- Mobile responsive navigation
